# ScheduLegend

ScheduLegend is a simple, no-signup alternative to When2Meet for scheduling meetings and finding a time that works for everyone.

Create a poll, share a link, and let participants mark their availability in a clear, easy-to-use grid.

## Features

- No signup required  
- Simple availability grid  
- Shareable links for participants  
- Works well on desktop and mobile  
- Create events using natural language (e.g. "next week", "July 2 to July 16")  

## Try it

👉 https://www.schedulegend.com

## When2Meet alternative

If you're looking for a When2Meet alternative, ScheduLegend offers a faster and more modern way to schedule group meetings. It keeps the familiar grid-based approach while making setup quicker and easier.

## Doodle alternative

ScheduLegend can also be used as a lightweight alternative to Doodle, especially if you want a simple, no-signup tool without unnecessary complexity.
